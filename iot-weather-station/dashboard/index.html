Real-Time Weather Dashboard

Raspberry Pi sensors stream temperature, humidity and air quality to a live dashboard over MQTT & Node-RED. Showing live Auckland weather by default — switch the source to your own hardware feed anytime.

LIVE · SIM
Real sensor data needs the Raspberry Pi and its sensors powered on and publishing to the MQTT broker. So this dashboard defaults to live Auckland weather — genuine temperature, humidity & PM2.5 from the free Open-Meteo API, no hardware required — and also includes a simulated feed for offline demos. Choose Live MQTT to stream from the physical hardware.
